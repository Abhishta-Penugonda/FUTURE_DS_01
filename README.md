# Business Sales Performance Analytics Dashboard

## Project Overview
This project focuses on analyzing business sales data to identify revenue trends, regional performance, customer segments, shipping modes, and top-performing product categories using Power BI.The dashboard provides interactive visualizations and business insights that help understand sales and profit performance across different regions and categories.

---
## Tools Used
- Power BI
- Excel / CSV Dataset
- GitHub
---

## Dataset Used
Sample Superstore Sales Dataset

---
## Dashboard Features
- Total Sales Analysis
- Profit Analysis
- Sales by Region
- Sales by Category
- Sales by Sub-Category
- Sales by Segment
- Sales by Payment Mode
- Monthly Sales Trend
- State-wise Sales Visualization
- Forecast Analysis
---

## Key Insights
- West region recorded the highest sales performance.
- Technology category generated high revenue and profit.
- Consumer segment contributed the highest share of sales.
- COD was the most preferred payment mode.
- Sales showed growth during year-end months.
---

## Recommendations
- Focus marketing strategies on high-performing regions.
- Improve performance in low-sales states.
- Increase stock availability for top-selling categories.
- Enhance shipping efficiency for better customer satisfaction.
